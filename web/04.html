ランディングページ制作

  • はじめての防衛白書（第２版）

現在参画しているプロジェクトで作成したランディングページです。

小学５年生以上をメインターゲットにしており、５年生で習わない漢字にのみルビを入れています。

小学生が夏休みの間に閲覧できるよう、８月頭には公開したいという要望があり確定の原稿が来る前にデザインを詰めておき、すぐに文章を流し込んで公開できるように調整しました。

小学生向けのコンテンツを作成するのは初めてだったので、小学生向けのホームページを検索して研究しました。ボタンのホバーを工夫しているホームページが多かったので、ホバーデザインをいくつか考え最終的には防衛省の方にデザインを決めていただきました。

担当領域 デザイン・コーディング
使用ツール Adobe Photoshop・Visual Studio Code
作成期間 1カ月
