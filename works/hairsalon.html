美容室

自主制作

https://mrsk613.github.io/hairsalon/

制作期間 2020年12月（工数：27時間ほど）
担当領域 デザイン／コーディング
ターゲット 20代後半から30代の男女
使用ツール figma／Adobe Photoshop／Visual Studio Code

自主制作のHPです。

20代後半から30代の男女をターゲットとして色味を当時流行っていたベージュで統一させて流行りの美容室感を出せるよう工夫しました。1ページのスクロールで完結するデザインにしたので、全体的な色味は抑えつつセクションごとに背景色を変えることで飽きずにスクロールできるよう工夫しました。

また、各セクションごとにデザインを変えつつフォントを英字のみ変更させて飽きない作りと全体の統一感を崩さないようにしました。
